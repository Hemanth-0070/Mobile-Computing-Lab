# Mobile Computing Lab Experiments

This repository contains Android application experiments developed using Android Studio and Java.

## Experiments

1. **Hello World Application**
   Basic Android application displaying "Hello World".

2. **Graphic Primitives using Canvas**
   Draws shapes like line, rectangle, circle, oval and text using Canvas.

3. **Image Slideshow**
   Displays images that change automatically every 3 seconds.

4. **Addition of Two Numbers**
   Simple Android app that takes two numbers and displays their sum.

## Tools Used

* Android Studio
* Java
* Android SDK
